# EmployeeManager

EmployeeManager is a basic C# console application for managing employees in memory.

## Features

- List all employees
- Add a new employee
- Search employees by id or name
- Update employee details
- Delete employee by id

## Project Structure

- `EmployeeManager/Program.cs`: Console UI and menu flow
- `EmployeeManager/Models/Employee.cs`: Employee model
- `EmployeeManager/Services/EmployeeService.cs`: In-memory employee operations

## Requirements

- .NET 10 SDK

## Run the app

From the repository root:

```bash
dotnet run --project EmployeeManager
```
